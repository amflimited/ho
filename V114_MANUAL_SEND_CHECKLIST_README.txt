v114 Manual Send Checklist / Compliance Review Layer

Purpose:
Adds a final manual pre-send checklist to Draft Ready cards without sending anything.

Adds:
- Manual Ready To Send visual label.
- Review Before Manual Send visual label when checklist fails.
- Display-only checklist:
  - public/customer-facing contact method
  - truthful subject
  - body references preview package without fake familiarity
  - no lead/ranking/sales guarantee
  - low-pressure language
  - asset links present
  - no SMS/call action
  - no unresolved warnings
- Copy To, Copy Subject, Copy Body buttons.
- Emphasized paused/manual-review warnings.

Important:
Manual Ready To Send is only a label/checklist state. There is no send action.

What v114 does NOT do:
- no email sending
- no SMS
- no AI calls
- no CRM automation
- no domain purchasing
- no upstream workflow changes

Suggested next step:
Stop and test the full downstream flow before adding more automation.
